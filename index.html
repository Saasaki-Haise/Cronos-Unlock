What is Cronos ?

Cronos is a EVM-compatible chain running in parallel with Crypto.org chain. Cronos delivers dramatically faster and cheaper transactions than Ethereum mainnet, making decentralized applications (DApps) and smart contracts more user and environmentally friendly. Built on Ethermint which is based on the Cosmos SDK, Cronos supports the rapid porting of DApps and smart contracts from Ethereum and other EVM-compatible chains, as well as interoperability with the Cosmos ecosystem via IBC (Inter Blockchain Communication).
